# jobsy_chrome_extension

Jobsy chrome_extension that allows the user to add jobs.
<br>
The user can apply to jobs in Linkedin and Indeed as usual.
<br>
When pressing on the apply button a popup will appear with the information about the role,company,location.
<br>
Sending the popup form will save the job in the server.
<br>
In addition user can highlight a text from any website and press on right click to save the text as role,company,location.
